Overview

After compiling the application using

go build github.com/hilverd/sitemapper

you can run

./sitemapper -help

to get a help message. Here is an example of how to (politely) crawl part of apple.com:

./sitemapper -v -max-depth 2 -max-concurrent-requests 8 apple.com | \
    tee apple-sitemap.txt
